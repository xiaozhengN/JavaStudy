1. 项目简单介绍: 
CDE 是通用桌面环境(Common Desktop Environment )的缩写。华为的 CDE 系统主要价值是：为全公司开发人员，提供一个通用的自动化办公的环境桌面，避免用户在多个系统间多次切换，提高开发人员办公效率。
2. 项目的技术栈:
项目前端采用 Vue ，后端采用 SpringBoot + SpringCloud + MybatisPlus ，数据持久化通过 MySQL 存储，
使用 Redis 做数据缓存，Kafaka 做消息中间件，项目发布底层采用 Jekins 做多节点自动发布。
3. 项目团队介绍:
按照业务发展，CDE 项目包含：注册中心(EurekaService)、网关(ApiGatewayService)、配置(ProjectConfigService)、数据(DateService)、代码(CodeService)、项目作业(ProjectWorkflowService)、P4Service(P4下载服务)、打点服务(PortalService)。总开发人数约60人。
4. 项目开发模式:
收集/挖掘用户诉求，转变为需求，大需求发现设计，详细设计方案、评审、开发、测试、发布。
5. 自己负责的模块
项目作业服务: 商用评审，商用合入看板。
代码服务: 文件外发、外非评审。
